# 문제 정의하고 해결하기

## 알고리즘이란

- 특정한 업무를 수행하기 위한 단계들의 집합

##

- 문제를 풀기위한 계획을 세운다
- 문제 풀이 패턴을 마스터한다

- 문제를 이해한다

## 1단계: 문제의 이해

- 문제를 자신만의 표현으로 바꾸자
- 문제에 입력하는 값과 출력값은 무엇인가?
- 출력값이 입력값에 의해서 결정이 되는가?
- 데이터의 중요한 이름을 무엇으로 붙여야할가?

## 2단계: 구체적인 예시를 찾자

- 예시는 최종 답안을 확인하는 데에 도움을 준다 (2~3개의 input과 output)
- 사용자가 유효하지 않은 값을 입력할 경우 생각
- 더욱 복잡한 예시 생각

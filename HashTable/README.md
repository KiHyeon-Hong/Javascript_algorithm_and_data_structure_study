# Hash table

- 카-값 쌍을 저장한다.
- 해시 테이블의 키는 순서가 없다.

# Hash function

- 키를 배열에서 사용되는 유효한 값으로 바꾸는 방법
- 임임의 크기의 데이터를 정해진 크기의 데이터로 변경
- 해시 값 자체로는 아무 의미가 없다.
